## Logo Usage Guidelines

### Formats Needed:
1. **Primary Logo** - Full color with text
2. **Icon Only** - For app icons, favicons
3. **Monochrome** - Single color versions
4. **Reversed** - For dark backgrounds

### Sizes:
- App Icon: 1024x1024px
- Favicon: 32x32px, 64x64px
- Social Media: 400x400px
- Website Header: SVG (scalable)
